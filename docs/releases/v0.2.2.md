## 新增
- Ollama 本地模型支持：可配置本地部署的 Ollama 服务（API 地址、密钥、模型名称），接入任意本地多模态模型
- AI 助手服务商/模型可配置：反推提示词和 Prompt 工程师功能现在可以选择使用 666API、巨游API 或 Ollama，并手动指定模型名称
- Prompt 工程师新增"中文输出"选项：勾选后强制使用简体中文输出提示词

## 优化
- 服务商设置页面新增 AI 助手服务商区域，统一管理反推提示词和 Prompt 工程师的模型配置

## 修复
- 修复 Ollama API 地址带 /v1 后缀时请求路径重复拼接导致 404 的问题
- 修复 Ollama 反推提示词无法识别本地文件路径格式图片的问题
- 修复 Rust 端中文响应日志截断时因多字节字符边界 panic 导致请求崩溃的问题
