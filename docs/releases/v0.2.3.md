# v0.2.3

## 新增功能

### 图片工具箱
- 新增独立的工具箱页面，从项目管理顶部进入
- **批量调整尺寸**：支持同时导出多个尺寸，可选像素/百分比模式，支持填充背景色、模糊背景、居中裁剪三种适配方式
- **图片裁剪**：可拖拽调整裁剪框大小和位置，支持锁定宽高比，显示三分构图辅助线
- **图片标注**：支持矩形、椭圆、箭头、画笔、文字标注，可自定义颜色和线宽
- **图片切割**：支持按行列数自动切割，或手动指定切割线位置

### 其他改进
- AI 图片节点默认记忆用户上次选择的供应商和模型
- 隐藏不常用供应商（KIE、派欧云、fal、GRSAI）
- 修复文字标注在 Tauri 中无法输入的问题
- 修复 Ollama 反推提示词中文截断崩溃问题
