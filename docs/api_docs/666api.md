### 核心接口（统一使用 OpenAI 兼容格式）

  

所有模型类型统一使用以下接口：

  

#### 1. 文本对话接口

```

POST https://www.666api.ai/v1/chat/completions
```

## 图像生成接口详解

  

### 方式一：Gemini 原生接口（推荐）

  

```

POST https://www.666api.ai/v1beta/models/gemini-3.1-flash-image-preview:generateContent

```

  

**请求示例**:

```json

{

"contents": [{

"parts": [{

"text": "Generate an image of a cute cat"

}]

}]

}

```

  

**测试结果**: ✅ 成功，生成图片并保存至本地

  

**图片保存方法**:

```python

import json

import base64

  

with open('cat_response.json', 'r') as f:

data = json.load(f)

  

inline_data = data['candidates'][0]['content']['parts'][0]['inlineData']

image_data = base64.b64decode(inline_data['data'])

  

with open('cat.png', 'wb') as f:

f.write(image_data)

```

  

**返回数据结构**:

```json

{

"candidates": [{

"content": {

"parts": [{

"inlineData": {

"mimeType": "image/jpeg",

"data": "<base64编码的图片数据>"

}

}]

}

}]

}

```

  

**生成的图片**: `e:\Code\666api\cat.png` (约 750KB)
